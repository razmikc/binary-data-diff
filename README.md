# Data Comparision

### Used technologies
* Java 11
* Spring Boot
* Spring MVC
* Spring Data JPA
* H2 database
* Swagger for Api Documentation
* Maven

### Requirements 

At least java 11 and Maven 3.5 are needed!

### Installation 

```sh
$ cd binary-data-diff
$ mvn clean package
$ java -jar target/data-diff-1.0.jar 
```

### Running tests

```sh
$ cd binary-data-diff
$ mvn test
```

### Api Documentation UI

[Swagger UI](http://localhost:8080/swagger-ui.html#)
